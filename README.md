# tum-phd-dissertation
This is an unofficial LaTeX template for TUM PhD dissertation title page based on the official [docx template](https://collab.dvb.bayern/download/attachments/69917104/20240419_Titelblatt.docx) provided by TUM graduation school.

## Usage
1. Clone this repository.
2. Edit the `main.tex` file to fill in your personal information and customize the title page as needed.
3. Compile the `main.tex` file using a LaTeX editor or command line to generate the PDF of your dissertation title page. Note that **ONLY** lulaTeX is supported for compilation due to the font requirements of the official TUM template.
